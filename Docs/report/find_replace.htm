Find and Replace

 

The Edit_Find and Replace menu function enables you to search for a word, part of a word or phrase in the currentreport and replace it with text that you specify.

 

If the search expression that you specify is found then the cursor moves to the first occurrence of this expression and it is highlighted/selected.
