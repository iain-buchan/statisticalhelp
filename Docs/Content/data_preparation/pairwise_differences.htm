Pairwise Differences

 

Menu location: Data_Transforming and Deriving_Pairwise_Differences.

 

This function creates a column of all possible (n*n) pairwise differences from a pair of columns:

All possible (n * n) Xi...(i=1 to k) are compared with Yj...(j=1 to k).

 

These contrasts form intermediate steps in a number of nonparametric statistical methods.
