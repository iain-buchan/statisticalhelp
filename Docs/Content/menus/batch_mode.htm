Batch Mode

 

When you are selecting data for a function in StatsDirect, a check box labelled "Batch Mode" may appear below the "OK" and "Cancel" buttons in the box at the top of the main StatsDirect window. If you click on "Batch Mode" a tick appears in its box, then, when you have finished analysing the current set of data the function for which you are selecting data will automatically activate again at the point where it asks you to specify the input data. Press the "Cancel" button to end a batch.
