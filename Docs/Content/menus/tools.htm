Tools

 

The Tools menu section contains links to external applications or other links such as Internet URLs.

 

There are five permanent tool links in the tools menu:-

  1. StatsDirect Calculator - evaluates formulae and can save the results to a report.
  2. StatsDirect Updates - opens the StatsDirect update web site if your web browser is working.
  3. Notepad - launches the Windows basic text editor.
  4. Wordpad - launches the Windows rich text editor.
  5. Windows Calculator - launches the Windows calculator.

 

In addition to the permanent links above, you can install your own links into the tools menu. Any command that works from the Windows run prompt will also work when run from the tools menu, for example you could enter the command line "mailto:sales@statsdirect.com" to create an email link to that address. See setup tools for more information on customizing the tools menu.
