Open File

 

The File_Open menu function enables you to open a StatsDirect workbook, a Microsoft Excel compatible workbook, a comma separated vector data file or a rich text file. Please note that there is also a direct data link with Microsoft Excel: see links with Excel for more information.

 

This function enables you to look for a file on a local storage medium or on a network, the place it starts looking is the location of the last file you opened.
