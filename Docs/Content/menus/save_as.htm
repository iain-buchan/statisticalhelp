Save As...

 

The File_Save As menu function saves the active StatsDirect data workbook or report to the location you specify using the file browser.

 

For data workbooks you can choose between the Excel 2007/2010 *.xlsx format or Excel 97/2000/XP/2003 *.xls for backwards compatibility. If you want to export the workbook to an application other than Excel then use the "Export data" function to export a tab delimited text file or a comma separated vector (CSV) file of the current worksheet in the workbook - these text file formats can be read by most applications.

 

For reports you can choose between: the default rich text format (RTF), which Microsoft Word or Wordpad can read; text file, where you will loose formatting; hypertext markup language (HTML), which web browsers can read; and MHTML, which is the same as HTML but packs all of the graphics files associated with a web page into a single file with the text.
