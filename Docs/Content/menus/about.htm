About StatsDirect

 

The Help_About StatsDirect menu function displays copyright and licence information for StatsDirect.

 

The platform that you are running StatsDirect from is also shown together with an approximate indication of the speed of the processor that is running.

 

Click on the StatsDirect update web site link if you want to see news or update information for StatsDirect. You must have a default web browser, such as Microsoft Internet Explorer, set up in Windows for this to work.
