Agreement Analysis

 

Menu location: Analysis_Agreement

  • Continuous (intra-class correlation etc.)
  • Categorical (kappa etc.)
  • Method Comparison Regression (Deming, Passing-Boblok etc.)
  • Reliability and Reducibility (principal components analysis and Cronbach's alpha)
  • Universal (Berry-Mielke R)

 

These methods look at the agreement of a set of measurements across a sample of individuals. Your data should be prepared so that each row represents data from an individual. Some literature lists these methods under 'reliability testing'.
