Cut

 

The Edit_Cut menu function deletes a selected block of text and copies it to the Windows clipboard. The same action can be done by holding down the control (Ctrl) key and pressing the X key.

 

You can retrieve the last cut text by selecting Paste from the edit menu or by holding down the control key (Ctrl) and then pressing the V key. Note that this will not work if you copied something else to the Windows clipboard after you copied the item(s) that you want to paste.
