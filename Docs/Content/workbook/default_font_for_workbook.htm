Default Font for Workbook

 

The Format_Default Font menu function enables you to specify the default font for the current workbook.

 

You can set a default font, font size, and style that will apply to all worksheets in the workbook. Later you can change these settings for individual cells in the Font tab of the Format_Cells dialog box. Changes you make in using Format_Cells will remain, even if you change the default settings.

 

The default font affects the widths of worksheet columns. Column widths are set in units equal to 1/256th of the character 0 (zero) in the default font. Because the basic unit for measuring columns can change when you change the default font, you may need to adjust the widths of columns (including the row header column) after setting the default font.
