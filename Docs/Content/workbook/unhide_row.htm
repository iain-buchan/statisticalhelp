Unhide Row

 

The Format_Row_Unhide menu function unhides a hidden row in a worksheet. You must first move to the hidden row in order to unhide it; hold down the Ctrl key and then press G to bring up the 'goto' dialog and then enter a cell reference that includes the row you want to unhide, e.g. enter A1 to go to row 1 or A8 to go to row 8.
