Worksheet Functions

 

StatsDirect gives you two ways to manipulate data in workbooks. The first is done via the "Data" menu where StatsDirect generates new workbook columns by manipulating existing ones, e.g. see apply function. The second method is to enter formulae in workbooks. This chapter describes most of the functions available in a Workbook. If you are already familiar with using functions in spreadsheets such as Microsoft Excel then StatsDirect workbook functions will be familiar to you.
