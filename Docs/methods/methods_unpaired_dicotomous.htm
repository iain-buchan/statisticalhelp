Advice: Fisher's Exact Test

 

Unpaired

Dichotomous

 

FISHER'S EXACT TEST: This is listed under the exact methods section of the analysis menu. There is little point in using a chi-squared test when this exact alternative is available. If you do use a chi-square test then all of your expected frequencies must be > 5. With large numbers you can use the methods for proportions which are listed under the proportions section of the analysis menu. The confidence interval for the comparison of two proportions can be useful with large numbers.
