Scatter Plot (Text-based)

 

Menu location: Graphics_Scatter (Text).

 

This function plots a Y axis (ordinate) variable against an X axis (abscissa) variable. The scale selection for the axes is automatic. Superimposed plot points are displayed as the number of plot points at one screen location provided this number is less than 10. If more than 9 plot points lie at one screen location then it is marked with the letter X. The graphics based version of this function allows you to display multiple Y series.
