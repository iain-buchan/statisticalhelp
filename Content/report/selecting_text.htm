You select a block of text in a Windows application by holding down the shift key and using the cursor keys to highlight the block of text you want. Alternatively, hold down the left mouse button and drag the mouse over the block of text you want to select.

 

The next action is usually to delete, cut or copy this block.

 

Key shortcuts  
Copy control (Ctrl) + C
Cut control (Ctrl) + X
Paste control (Ctrl) + V
