Importing Data

 

The File_Import Data menu function imports plain text files (tab or comma and quote delimited) into the active worksheet of the current workbook. Most spreadsheets can save worksheets in one of these text-based formats.

 

A common text format is the CSV (comma separated vector) file. You can open a CSV file directly into StatsDirect using the "File Open" menu function.

 

You can also read Microsoft Excel workbooks directly into StatsDirect via the "File Open" menu function. If you are importing data from Microsoft Excel then please see Excel links.

 

You can also use the windows clipboard to cut and paste selected cells or complete worksheets between Windows applications that use standard worksheet formats.
