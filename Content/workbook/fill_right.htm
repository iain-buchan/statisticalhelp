Fill Right

 

The Edit_Fill Right menu function causes all columns of a selected workbook block to be filled with the contents of the left hand column, i.e. this is "copied right".

 

You can achieve the same effect by using the mouse only: hold down the left mouse button when the mouse cursor is over the bottom right hand corner of the box around the active cell and drag across a rectangular selection to the point you want to copy to on the right hand side.

 

Note that in some applications this type of function is called 'Copy right'.
