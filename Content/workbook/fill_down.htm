Fill Down

 

The Edit_Fill Down menu function causes all rows of a selected block in a worksheet to be filled with copies of the first row, i.e. the first row is "copied down".

 

You can achieve the same by using the mouse only: hold down the left mouse button when the mouse cursor is over the bottom right hand corner of the box around the active cell and drag down a rectangular selection to the point you want to copy to.

 

Note that in some applications this type of function is called 'Copy down'.
