Paste Special

 

The Edit_Paste Special menu function pastes the content that you choose from selected cells from the Windows clipboard into the active worksheet of the current workbook. You select pasting of values only (i.e. the result of a formula in a cell is pasted and not the formula), formulae, format only or all content (same as paste function).

 

You can also use the windows clipboard to cut and paste selected cells or complete worksheets between applications that use standard formats for worksheets, e.g. Microsoft Excel.
