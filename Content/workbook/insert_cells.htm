Insert Cells

 

The Insert_Cells menu function provides different options for inserting rows, columns or blocks of cells in the active worksheet of the current workbook.

 

If you want to insert a block of blank cells into part of a worksheet with data at that location then select the block you want and use this menu function. When a block of cells is inserted, the existing cells at the selected location are either shifted down or to the right according to the direction you specify.
