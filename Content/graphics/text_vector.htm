StatsDirect provides text-based versions of some plots for rapid transmission as 7 bit ASCII text. The graphical versions of these plots are intended for routine use / presentation. StatsDirect graphics use the Windows metafile format to achieve high quality when printing from high resolution printers with good Windows drivers. You can paste StatsDirect graphs to other applications, such as Microsoft Word or PowerPoint, using the Windows clipboard (Ctrl+C to copy in a StatsDirect report then Ctrl+V to paste to a Word document or PowerPoint presentation). If you then double click on the graphic in Word or PowerPoint, or right click and select "Edit Picture", you can edit it.
